# NyxOS Mint Roadmap

Goal: evolve NyxOS into a stable, lightweight desktop OS with the user experience, tools, and philosophy of Linux Mint Cinnamon.

Subgoals:
- 1.1 Advanced window management
- 1.2 Taskbar + Mint-style start menu + system tray + clock
- 1.3 GUI file manager
- 1.4 SETTINGS/CONTROL CENTER
- 1.5 X-APPS: TEXT EDITOR, IMAGE VIEWER, AUDIO PLAYER, GUI TERMINAL
- 2.1 Update manager
- 2.2 Package manager (.pkg)
- 3.1 EXT2 read/write
- 3.2 ELF loader + initramfs
- 3.3 ATA/IDE driver
- 4.1 SB16 audio integration
- 5.1 Theme and icons
- 5.2 Defaults
- 5.3 Help system
- 5.4 Keep KISS philosophy
